How To Use Biblio

Quick guide for users!

How to add books?

Just click the +Add button in the menu, then enter the 13 digits of isbn code.

Check the content and click add book button!

How to borrow books?

Click the request button of the book in a homepage. The request mail will be automatically sent out to the book's owner so that you can communicate with owner where and when to borrow the book.

After you borrowed the book, click borrow button. You can see all books you borrowed in a borrowed page

How to return books?

Got to the borrowed books page, then click the request button of the book. The request mail will be automatically sent out to the book's owner so that you can communicate with owner where and when to return the book.
